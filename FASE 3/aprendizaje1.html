Etiquetas Semanticas

las etiquetas semánticas sirven para definir las diferentes estructuras de un documento HTML, la cual se le incorporan diversas técnicas para el diseño del sitio web

También podemos concluir que las etiquetas semánticas llevan en su documento una estructura básica, como lo es su cabecera, un cuerpo y un pie de página.

Las etiquetas semánticas las puede desarrollas cualquier sistema operativo, brindando todas las herramientas necesarias para el usuario.

CSS Grid

Se puede concluir que el CSS Grid, ofrece diferentes opciones para diseñar diferentes diseños.

También se puede concluir que el CSS Grid, permite alinear y configurar las estructuras de las etiquetas semánticas.

Los diseños de sitios web tienen éxito en su creación, gracias a las diferentes herramientas que el CSS Grid ofrece al diseñador
