CSS Grid

Lenguajecss.com (2020) Manz, Grid css “cuadricula” recuperado de: https://lenguajecss.com/css/maquetacion-y-colocacion/grid-css/grid-template-areas.png/

CSS Grids nos permite generar layouts o distribuciones de contenido en nuestro sitio web de forma eficiente mediante cuadriculas compuesta por filas y columnas. Para disponer de todas las características que nos brinda CSS Grid primero debemos definir un Grid Container, es decir, definirle a una etiqueta que haga la función de contenedor en nuestro HTML un display de tipo grid. Cada uno de sus hijos directores (Grid Items) compondrán la cuadricula, lo que nos permitirá alinearlos a nuestro antojo
